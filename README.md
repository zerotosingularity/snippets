# Python

## Opencv version
```bash
import cv2
vc2.__version__
```

## Pip package version
```bash
pip list | grep the_pip_package_you_seek
```

# Bash
## Cuda version
```bash
nvcc --version
```

# Tmux

## list sessions
```bash
tmux list-sessions
```

## attach to session
```bash
tmux attach-session -t #
```

## split vertically
```bash
Ctrl+b %
```

## split horizontal
```bash
Ctrl+b "
```

## cycle through panes
```bash
Ctrl+b o
```

## zoom to current pane
```bash
Ctrl+b z
```

# Linux
## Print linux version info

```bash
cat /etc/*-release
```

## kill pane
```bash
Ctrl+b x
```
